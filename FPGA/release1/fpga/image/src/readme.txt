## Device Tree Compiler (DTC)
To convert device-tree in dts format into dtb you need dtc program.
It can be found at git://github.com/Xilinx/linux-xlnx.git (inside scripts/dtc/ directory)

DTC is dependent on Linux kernel!!

## Patching
To have correct ethernet phy ID, added EEPROM and working XADC in PL you need to apply patch.
